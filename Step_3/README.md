- 필요성:
    - 네트워크 통신을 통해서 데이터를 교환하고 받는 것은 중요하다. 이번 스텝에서는 시뮬레이터 컴퓨터와 원격 컴퓨터간

    5. logging 라이브러리를 사용해서 프로그램의 log를 저장한다.
